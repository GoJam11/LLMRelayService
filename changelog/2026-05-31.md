# 2026-05-31

### 新增

- 为 managed API key 增加累计费用额度：新增费用额度字段，额度按请求成本扣减，管理员环境变量 key 不受分发额度限制。
- 网关在 managed API key 达到费用额度后返回 429。
- 控制台 Keys 页面增加费用额度展示和编辑弹窗，支持设置非负美元金额，留空表示不限制。
- OpenAPI 和控制台 API 增加 `PATCH /keys/:id/quota` 管理接口。

### 变更

- 将上一版 Token 额度语义更正为费用额度语义，DB 内部使用 micro-USD 整数保存累计费用，避免浮点累计误差。
