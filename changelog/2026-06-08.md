# 2026-06-08

### 新增
- Routes 页面新增路由地图，按 request model 展示实际首选真实路由、当前回退策略下可能追加的真实模型，以及 alias 覆盖的同名模型，禁用渠道和禁用别名不纳入展示。
- 新增长期虚拟模型路由能力：Provider 支持 `direct` / `explicit_only` 路由可见性，Routes 支持一个虚拟模型配置多个后端 targets，可将 `gpt-5.5-third` 显式路由到廉价供应商的真实 `gpt-5.5`。

### 变更
- 统一 Routes 页面路由地图表格样式，将路由项改为 Badge 展示并补齐来源标签国际化，使其与 Console 其他表格的文字层级和组件风格一致。
- 修复 Console 夜间模式下背景、主内容 surface 和表格 shell 仍使用浅色硬编码的问题，并让 Routes 路由地图使用统一表格暗色样式。
- 普通客户端模型解析、`/v1/models` 和全局 same/any fallback 只使用 `direct` Provider；`explicit_only` Provider 仅能通过虚拟模型 targets 或自定义 fallback 的 `provider:model` 显式引用。
