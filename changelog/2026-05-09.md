# 2026-05-09

### 变更
- `docker-compose.yml`：将 `app` 服务从本地 `build: .` 改为直接拉取 GHCR 预构建镜像 `ghcr.io/gojam11/llmrelayservice:main`，解决用户 Docker 部署时 build 上下文缺失或环境差异导致的页面异常问题。

- `README.md`：将部署章节拆分为「Docker Compose 部署（推荐）」和「单容器 Docker 部署」，补充 GHCR 镜像地址、端口说明和 `docker compose pull` 更新命令。

### 修复
- 渠道页面「导出」「导入」按钮显示原始 i18n key `common.export` / `common.import`：在 `zh.ts` 和 `en.ts` 的 `common` 命名空间中补全对应翻译条目。
- 渠道页面导入/导出弹窗：补齐渠道配置相关的中英文 i18n 文案，并为弹窗设置视口限高和固定 JSON 编辑区，避免配置内容过多时窗口超出屏幕。
