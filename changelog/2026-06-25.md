# 2026-06-25

### 新增
- 模型路由别名新增「返回真实模型名」开关（默认关闭）。关闭时网关会把上游响应里的 `model` 字段从真实模型名（如 `gpt-4o-mini`）改写回 alias 名（如 `Auto`），覆盖 JSON 和 SSE 两种格式，并跨 chunk 边界安全替换；启用时则保持上游返回的真实模型名透传。
  - DB schema 新增 `model_aliases.return_real_model` 列（drizzle 迁移 `0008_alias_return_real_model.sql`）。
  - `console-model-alias-store` / `config.ts` 加 `returnRealModel` 字段，alias 解析得到的 `RouteResult` 会带上该标记。
  - `response-observer.ts` 新增 `createModelFieldRewriteBody`，作为 TransformStream 接入到 `finalizeProxyResponse`，索引到所有走 alias 路由的成功响应。
  - 控制台「路由策略」表单新增 Checkbox（zh/en 文案 `routes.returnRealModel*`），列表里启用「回传真实模型名」时显示 outline badge 提示。
- 新增单元测试 `test/response-model-rewrite.test.ts`，覆盖 JSON、SSE 多块分包、from==to 等场景。
