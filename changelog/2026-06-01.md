# 2026-06-01

### 变更

- 将 README 顶部控制台展示图从三张旧截图替换为单张新版 `docs/screenshots/lrs.png`。
- 拆分普通请求与流式请求的上游首字节超时：普通请求默认 FTTB 调整为 300000 毫秒，流式请求新增独立 `UPSTREAM_STREAM_FIRST_BYTE_TIMEOUT_MS` 默认 30000 毫秒，并在控制台运行配置中单独展示和保存。
- 恢复 `UPSTREAM_REQUEST_TIMEOUT_MS` 作为首字节超时旧配置名 fallback，避免已有环境变量配置在拆分后失效。
- Routes 页全站回退策略支持在自定义 fallback 列表中填写路由别名或 `渠道名:模型名`，不再把不同渠道的同名模型视为同一个 fallback 目标；alias 作为虚拟模型拥有独立模型级回退配置，不继承其真实模型的回退规则；并新增公共帮助弹窗组件说明自定义回退、alias 回退和尝试顺序。
