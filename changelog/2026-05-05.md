# 2026-05-05

### 新增
- 创建渠道弹窗支持「Sync」按钮拉取上游模型列表（无需先保存渠道）
  - 前端：移除 `dialogMode === 'edit'` 限制，创建模式下也可点击 Sync
  - 前端：Sync 按钮在 `targetBaseUrl` 为空时自动禁用
  - 后端：`/upstream-models-preview` 接口已存在，支持用表单参数直接请求上游 `/models`
  - i18n：更新 `syncNeedUrl` 提示，不再强制要求填写 Credential
- Providers 页面支持配置导入/导出
  - 导出：生成含 `version: 1` 和 `exportedAt` 时间戳的 JSON，包含所有 provider 配置（含 models、auth、extraFields 等）
  - 导入：解析 JSON 并逐个调用 `createProvider` 创建，支持版本校验（仅接受 version 1）
  - 新增 i18n 文案：`importConfigTitle`、`exportConfigTitle`、`importConfigDesc`、`exportConfigDesc`、`importConfigPlaceholder`、`importConfigEmpty`、`importConfigVersionMismatch`、`importConfigInvalid`、`importConfigConfirm`
